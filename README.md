## Overview

BalirajaHub is a bilingual smart agriculture platform designed to simplify farming through digital technology.

It provides tools for:

- Farm Management
- Crop Management
- Weather Insights
- Government Schemes
- Marketplace
- Expense Tracking
- AI Assistant
- Disease Detection
- Inventory Management

## Tech Stack

### Backend

- Java 21
- Spring Boot
- Spring Security
- Spring Data JPA
- JWT
- MySQL
- Maven

### Frontend

- React
- Tailwind CSS
